# @fgv/chocolate-cli

Command-line interface for chocolate recipe management.

## Overview

This CLI tool provides commands for managing chocolate recipes, including:
- Ingredient database management
- Recipe scaling and validation
- Recipe analysis

## Status

🚧 Under active development - Stage 1 (Package Setup)

## Installation

```bash
rush build
```

## Usage

```bash
choco --help
```

Commands will be added as functionality is implemented.

## Development

```bash
# Build
rushx build

# Test
rushx test

# Lint
rushx lint
```
