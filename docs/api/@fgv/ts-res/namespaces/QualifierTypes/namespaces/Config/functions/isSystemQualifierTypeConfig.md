# Function: isSystemQualifierTypeConfig()

> **isSystemQualifierTypeConfig**(`config`): `config is ISystemQualifierTypeConfig`

Checks if a [qualifier type configuration](../type-aliases/IAnyQualifierTypeConfig.md) is a
[system qualifier type configuration](../type-aliases/ISystemQualifierTypeConfig.md).

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `config` | [`IAnyQualifierTypeConfig`](../type-aliases/IAnyQualifierTypeConfig.md) | The [qualifier type configuration](../type-aliases/IAnyQualifierTypeConfig.md) to check. |

## Returns

`config is ISystemQualifierTypeConfig`

`true` if the configuration is a system qualifier type configuration, `false` otherwise.
