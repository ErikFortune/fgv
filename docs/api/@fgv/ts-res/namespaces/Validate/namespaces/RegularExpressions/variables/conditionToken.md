# Variable: conditionToken

> `const` **conditionToken**: `RegExp`

**`Internal`**

A [condition token](../../../../../type-aliases/ConditionToken.md) is a comma-separated list of one or more
condition specifiers, where a condition specifier has either the form
`<qualifierName>=<value>` or `<value>`.
