# Variable: qualifierDefaultValueToken

> `const` **qualifierDefaultValueToken**: `RegExp`

**`Internal`**

A [qualifier default value token](../../../../../type-aliases/QualifierDefaultValueToken.md) has the form
`<qualifierName>=<value>` or `<qualifierName>=` (to remove default).
Default values allow broader character set including commas for comma-separated values.
