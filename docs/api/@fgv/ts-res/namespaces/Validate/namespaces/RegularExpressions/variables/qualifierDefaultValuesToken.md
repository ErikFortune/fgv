# Variable: qualifierDefaultValuesToken

> `const` **qualifierDefaultValuesToken**: `RegExp`

**`Internal`**

A [qualifier default values token](../../../../../type-aliases/QualifierDefaultValuesToken.md) is a
pipe-separated list of one or more qualifier default value tokens.
Uses "|" as separator to avoid conflicts with comma-separated values within defaults.
