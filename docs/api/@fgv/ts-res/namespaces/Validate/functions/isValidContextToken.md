# Function: isValidContextToken()

> **isValidContextToken**(`token`): `token is ContextToken`

Determines whether a string is a valid context token.
Context tokens are pipe-separated lists of context qualifier tokens.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `token` | `string` | the string to validate. |

## Returns

`token is ContextToken`

`true` if the string is a valid context token, `false` otherwise.
