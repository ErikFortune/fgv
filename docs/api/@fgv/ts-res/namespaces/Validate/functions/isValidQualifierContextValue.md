# Function: isValidQualifierContextValue()

> **isValidQualifierContextValue**(`value`): `value is QualifierContextValue`

Determines whether a string is a valid qualifier context value.
For now, any non-empty string is considered a valid context value.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `value` | `string` | the string to validate |

## Returns

`value is QualifierContextValue`

true if the string is a valid qualifier context value, false otherwise.
