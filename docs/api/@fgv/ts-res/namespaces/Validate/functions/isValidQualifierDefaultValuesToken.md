# Function: isValidQualifierDefaultValuesToken()

> **isValidQualifierDefaultValuesToken**(`token`): `token is QualifierDefaultValuesToken`

Determines whether a string is a valid qualifier default values token.
Qualifier default values tokens are pipe-separated lists of qualifier default value tokens.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `token` | `string` | the string to validate. |

## Returns

`token is QualifierDefaultValuesToken`

`true` if the string is a valid qualifier default values token, `false` otherwise.
