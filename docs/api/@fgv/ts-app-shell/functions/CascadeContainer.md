> **CascadeContainer**(`props`): `ReactElement`\<`unknown`, `string` \| `JSXElementConstructor`\<`any`\>\> \| `null`

Horizontal scroll container for the column cascade.

detail columns below. Auto-scrolls to the rightmost column when
