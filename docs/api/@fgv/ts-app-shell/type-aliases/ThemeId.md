# Type Alias: ThemeId

> **ThemeId** = `string`

A theme identifier string. Built-in values are 'light', 'dark', and 'system'.
Custom themes use arbitrary strings that map to CSS classes (e.g., 'ocean' maps to '.theme-ocean').
