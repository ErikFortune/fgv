# Interface: IGridPresentationOptions

Presentation options for grid display.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="classname"></a> `className?` | `string` | Additional CSS classes for the grid container |
| <a id="enablefiltering"></a> `enableFiltering?` | `boolean` | Enable filtering of grid rows |
| <a id="enablesorting"></a> `enableSorting?` | `boolean` | Enable sorting of grid rows |
| <a id="pagesize"></a> `pageSize?` | `number` | Number of rows per page (0 for no pagination) |
| <a id="showrownumbers"></a> `showRowNumbers?` | `boolean` | Whether to show row numbers |
| <a id="showsummaryrow"></a> `showSummaryRow?` | `boolean` | Whether to show a summary row |
