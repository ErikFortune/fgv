# Function: isEncryptedFile()

> **isEncryptedFile**(`json`): `boolean`

Checks if a JSON object appears to be an encrypted file.
Uses the format field as a discriminator.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `json` | `unknown` | JSON object to check |

## Returns

`boolean`

true if the object has the encrypted file format field
