## IJsonConverterOptions.useConditionalNames property

If `true` and if template variables are available,
then string property names will be considered for
conditionals.

Default is to match IJsonConverterOptions.useNameTemplates | useNameTemplates.

**Signature:**

```typescript
useConditionalNames: boolean;
```
