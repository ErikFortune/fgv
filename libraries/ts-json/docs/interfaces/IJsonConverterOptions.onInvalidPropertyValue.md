## IJsonConverterOptions.onInvalidPropertyValue property

If IJsonConverterOptions.onInvalidPropertyValue | onInvalidPropertyValue is `'error'`
(default) then any illegal property value causes an error and stops conversion.  If
IJsonConverterOptions.onInvalidPropertyValue | onInvalidPropertyValue is `'ignore'` then
any invalid property values are silently ignored.

**Signature:**

```typescript
onInvalidPropertyValue: "error" | "ignore";
```
