# Type Alias: MeasurementUnit

Measurement unit types for recipe ingredients.
- 'g': Grams (default)
- 'mL': Milliliters
- 'tsp': Teaspoons
- 'Tbsp': Tablespoons
- 'pinch': A small imprecise amount
- 'seeds': Individual seeds (e.g., vanilla seeds)
- 'pods': Whole pods (e.g., vanilla pods)

## Type

```typescript
type MeasurementUnit = "g" | "mL" | "tsp" | "Tbsp" | "pinch" | "seeds" | "pods"
```
