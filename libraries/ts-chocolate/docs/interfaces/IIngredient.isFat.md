## IIngredient.isFat() method

Returns true if this is a fat ingredient.
When true, fat-specific properties are available.

**Signature:**

```typescript
isFat(): this is IFatIngredient;
```

**Returns:**

this is [IFatIngredient](IFatIngredient.md)
