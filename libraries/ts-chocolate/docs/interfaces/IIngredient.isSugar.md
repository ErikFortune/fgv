## IIngredient.isSugar() method

Returns true if this is a sugar ingredient.
When true, sugar-specific properties are available.

**Signature:**

```typescript
isSugar(): this is ISugarIngredient;
```

**Returns:**

this is [ISugarIngredient](ISugarIngredient.md)
