## IConfectionMarkdownOptions.variations property

Which variations to render.
- `'golden'` (default): only the golden variation
- `'all'`: every variation
- `ConfectionRecipeVariationSpec[]`: explicit list

**Signature:**

```typescript
readonly variations: "all" | "golden" | ConfectionRecipeVariationSpec[];
```
