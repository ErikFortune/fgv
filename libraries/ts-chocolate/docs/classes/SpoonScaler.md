# Class: SpoonScaler

Scaler for spoon measurements (tsp/Tbsp) with fractional support.
Handles conversion between tsp and Tbsp (1 Tbsp = 3 tsp).

**Implements:** [`IUnitScaler`](../interfaces/IUnitScaler.md)
