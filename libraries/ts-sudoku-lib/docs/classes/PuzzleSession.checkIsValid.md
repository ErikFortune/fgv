## PuzzleSession.checkIsValid() method

Determines if the puzzle is valid in its current state.

**Signature:**

```typescript
checkIsValid(): boolean;
```

**Returns:**

boolean

`true` if all non-empty cells in the puzzle are valid,
or `false` if any cells are invalid.
