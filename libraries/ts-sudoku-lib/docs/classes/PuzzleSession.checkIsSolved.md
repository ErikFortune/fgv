## PuzzleSession.checkIsSolved() method

Determines if the puzzle is correctly solved.

**Signature:**

```typescript
checkIsSolved(): boolean;
```

**Returns:**

boolean

`true` if the puzzle is solved, `false` if the puzzle has
empty or invalid cells.
