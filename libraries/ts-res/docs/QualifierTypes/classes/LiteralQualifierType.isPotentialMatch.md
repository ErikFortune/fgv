## LiteralQualifierType.isPotentialMatch() method

Determines if a supplied condition value is a potential match for a possible context value.

**Signature:**

```typescript

```

**Parameters:**

<tr><td>conditionValue</td><td>string</td><td>The condition value.</td></tr>
<tr><td>contextValue</td><td>string</td><td>The context value.</td></tr>
</tbody></table>

**Returns:**

boolean

`true` if the condition value is a potential match for the context value, `false` otherwise.
