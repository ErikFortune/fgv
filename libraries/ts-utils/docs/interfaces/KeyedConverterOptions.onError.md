## KeyedConverterOptions.onError property

if `onError` is `'fail'` (default), then the entire conversion fails if any key or element
cannot be converted.  If `onError` is `'ignore'`, failing elements are silently ignored.

**Signature:**

```typescript
onError: "fail" | "ignore";
```
