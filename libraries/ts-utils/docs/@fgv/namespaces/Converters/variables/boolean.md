# Variable: boolean

> `const` **boolean**: [`Converter`](../../Conversion/interfaces/Converter.md)\<`boolean`, `unknown`\> = `booleanConverter`

A [Converter](../../Conversion/interfaces/Converter.md) which converts `unknown` to `boolean`.

## Remarks

Boolean values or the case-insensitive strings `'true'` and `'false'` succeed.
Anything else fails.
